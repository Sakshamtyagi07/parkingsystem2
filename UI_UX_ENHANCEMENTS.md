## üí° CUSTOMIZATION

To adjust animations:

1. **Speed**: Edit animation duration (e.g., `8s` ‚Üí `5s`)
2. **Colors**: Modify gradient colors in keyframes
3. **Effects**: Add/remove animations in selectors
4. **Intensity**: Adjust blur amount, shadow spread

Example:
```css
/* Slower animation */
@keyframes gradientShift {
  /* Change from 15s to 20s */
  animation: gradientShift 20s ease infinite;
}
```
